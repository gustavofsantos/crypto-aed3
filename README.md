# Trabalho de AED3

### Dependências

É necessário:
* Usar Linux
* Ter o OpenSSL instalado
* Python instalado (pode ser 2 ou 3)


### Como executar

```bash
$ python gen.py <combinação inicial> <combinação final>
```

Exemplo:

```bash
$ python gen.py aaaa f999
```

*Aviso:* Ctrl+C não funciona, use Ctrl+Z para parar a execução
